# 什么是 Openmim？

Openmim 是一个基于 **python-telegram-bot** 构建的、充满个性的 Telegram 群聊机器人框架。它不是普通的问答机器人，而是一个能够自我进化、主动参与群聊、拥有持久记忆的 AI 聊天伙伴。

## 设计理念

大多数 Telegram AI 机器人存在两个核心问题：

1. **"机器感"太强** —— 每次都能感觉到在和机器说话，句式固定，AI味浓重
2. **回复时机生硬** —— 随机触发或只响应 @，完全无法融入群聊节奏

Openmim 通过一系列精心设计解决了这两个问题：

- 🎯 **聚焦系统**：LLM 实时评分每条消息，只在真正"感兴趣"时才主动插话
- 🎭 **拟人化引擎**：微动作、打字节奏、去AI味后处理，让回复更像真人
- 🧠 **三层记忆**：群聊上下文、用户人格画像、机器人自身人格——随时间进化

## 核心功能一览

| 功能 | 说明 |
|------|------|
| **自适应记忆** | 群聊记忆 + 用户画像 + 机器人人格，持续学习和进化 |
| **聚焦插话** | LLM 双阶段评分，见缝插针参与群聊，或被关键词/昵称召唤 |
| **Agentic 工具** | 联网搜索、代码执行、图片生成、计划任务、Skills 市场 |
| **Business 模式** | 私聊接管，以自定义身份代替你与用户对话 |
| **Guest 模式** | 任意群组无需白名单即可召唤 |
| **分群 BYOK** | 每群独立 LLM/图生 API 配置 |
| **Web 管理面板** | FastAPI 驱动的可视化管理界面 |
| **Telegram 面板** | 无需打开浏览器即可管理群设置 |
| **拟人化** | 微动作、情绪、打字节奏、去AI味 |
| **可玩性** | 每日运势、看图猜时代、复读机、定时问候 |

## 技术栈

- **运行时**：Python 3.11+
- **Telegram 框架**：[python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) 22.x
- **数据库**：SQLite（通过 SQLAlchemy ORM）
- **LLM 接入**：OpenAI 兼容 API（支持任何 OpenAI 格式的服务商）
- **Web 框架**：FastAPI + Uvicorn
- **代码沙箱**：E2B / Shipyard / Local

## 适合谁使用？

- 希望给群组加入一个有个性、有记忆、会主动聊天的 AI 伙伴
- 想用 Telegram 机器人接管私聊的个人/小团队
- 希望完全掌控 LLM 接入、自定义人设和行为的开发者

::: tip
Openmim 的所有 LLM 调用都基于 OpenAI 兼容接口。你可以使用 OpenAI、Anthropic（通过中转）、Gemini（通过中转）、DeepSeek、本地 Ollama 等任意服务。
:::
