# 管理面板

Openmim 提供两种管理方式：Telegram 内联面板和 Web 面板。

## Telegram 管理面板

所有管理操作均可在 Telegram 中直接完成，无需打开浏览器。

### 管理员面板（`/admin`）

**权限**：需要在 `ADMIN_IDS` 中配置

**功能模块**：

#### 白名单管理
- 查看所有白名单群组列表
- 添加群组（输入群组 ID）
- 移除群组

#### 全局状态
- 查看机器人状态
- 查看白名单统计
- 查看已加载贴纸

#### 记忆管理（全局）
- 查看机器人人格记忆列表
- 添加记忆条目
- 删除记忆条目

#### 插件状态
- 查看所有插件的启用状态

---

### 群管理面板（`/gadmin`）

**权限**：需要是该 Telegram 群组的管理员

**在哪里使用**：在群聊中发送 `/gadmin`

**功能模块**：

#### 基础设置
- 开关早安/晚安问候
- 开关冷群话题激活
- 开关自由回复模式
- 设置消息丢弃概率
- 开关用户名脱敏
- 开关复读机

#### 人设与评分
- 自定义群专属人设
- 自定义聚焦评分标准
- 设置注意力模式
- 设置回复偏好

#### 接口配置（BYOK）
- 配置独立 LLM 模型/API
- 配置独立图片生成 API
- 配置独立 Tavily 搜索 Key

#### Skills 管理
- 浏览可用 Skills 列表
- 订阅/取消订阅 Skills
- 配置 Skill 密钥

#### 工具管理
- 查看可用工具列表
- 禁用/启用特定工具

---

### Business 设置面板（`/settings`）

**权限**：私聊机器人的任意用户（Business 模式中为所有者）

**功能**：
- 设置机器人代表的名字
- 自定义人设 prompt
- 管理同义词（说话风格学习）

---

## 常用管理操作

### 添加群组到白名单

**方式一**：私聊机器人 `/admin` → 白名单管理 → 添加群组

**方式二**：在目标群中向机器人发送群组 ID（需要机器人已在该群）

**获取群组 ID**：
- 将 [@userinfobot](https://t.me/userinfobot) 添加到群组，让它报告群组 ID
- 或者查看 Telegram 链接（`https://t.me/c/1234567890/1` 中的 `1234567890` 加上 `-100` 前缀）

### 给机器人静音（针对特定用户）

任意用户可以使用 `/muteme` 让机器人忽略自己的消息：

```
/muteme       # 机器人不再回复我
/unmuteme     # 恢复回复
```

### 查看机器人关于信息

```
/about        # 显示机器人信息、版本等
```

## 面板操作技巧

### 导航

Telegram 内联面板使用按钮导航。如果面板过期（Telegram 限制），重新发送命令即可刷新。

### 文本输入

部分操作需要输入文本（如设置人设、填写 API Key），系统会提示"请发送..."，直接在群聊中回复即可。

输入 `/cancel` 可取消当前操作。

### 批量操作

对于需要批量管理的操作（如管理大量群组的白名单），建议使用 Web 面板，操作更方便。
