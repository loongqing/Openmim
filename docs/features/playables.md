# 可玩性功能

Openmim 内置了多种有趣的互动功能，让群聊更加生动活泼。

## 每日运势 `/fortune`

机器人为用户生成每日运势，每天每人只能查询一次（UTC 0 点重置）。

运势包含：
- 今日运势总评（大吉/吉/平/凶等）
- 幸运色/幸运数字
- 今日宜/忌
- 一句个性化的运势解读

运势由 LLM 生成，每次都不相同。

**使用方法**：
```
/fortune        # 查看自己的运势
```

## 看图猜时代 `/guesshistory`

一个益智游戏，机器人从网络搜索历史照片，让群成员猜测照片拍摄的年代和地区。

**游戏流程**：
1. 发送 `/guesshistory` 或"猜时代"等触发词
2. 机器人发布一张历史照片
3. 群成员自由猜测年代和地点
4. 一段时间后可以用"公布答案"等触发词揭晓答案
5. LLM 会分析群成员的猜测并给出评论

**触发词**：
- 开始：`猜时代`、`猜历史`、`开始猜图`、`历史猜图`
- 揭晓：`公布答案`、`显示答案`、`揭晓`、`看答案`

**地点范围**：
覆盖全球 50+ 个历史城市，包括：
- 欧洲：巴黎、伦敦、柏林、维也纳等
- 亚洲：东京、上海、北京、孟买等
- 美洲：纽约、芝加哥、里约热内卢等

**时代范围**：约 1850-1980 年的历史照片

::: tip 自定义触发词
可以通过 `data/customization.json` 自定义触发词列表。
:::

## 复读机

当群里出现连续复读时，机器人会以一定概率：
- 跟着一起复读
- 吐槽复读行为（"又来了"、"停不下来了"等）
- 接一句相关的话

这个功能可以在群设置中通过 `repeater_enabled` 控制。

## 定时问候

机器人每天在固定时间向白名单群组发送问候消息：

### 早安（08:00 CST）

机器人在早上 8 点向开启了早安功能的群组发送早安消息。

### 晚安（23:00 CST）

机器人在晚上 11 点向开启了晚安功能的群组发送晚安消息。

问候内容由 LLM 根据贴纸库随机生成，可能附带贴纸。

**群组控制**：
在群管理面板（`/gadmin`）中可以分别开关早安/晚安功能。

## 机器人昵称召唤

除了通过 @ 触发，用户还可以直接在消息中使用机器人的昵称来召唤它。

**默认昵称**：
- 猫猫
- 小猫  
- 咪姆
- 咪姆酱

**自定义昵称**：
```json
{
  "BOT_CALL_ALIASES": ["猫猫", "小猫", "咪姆", "自定义名字"]
}
```

## 自由回复模式

开启群设置中的 `free_reply_mode` 后，机器人会进入自由回复模式：

- Bot 可以看到最近消息的 ID 列表
- LLM 可以选择性地对一条或多条消息分别回复
- 已回复的消息会被标记，避免重复追回同一条消息

这让机器人的回复更加灵活自然，适合活跃的群聊环境。

## 消息分段发送

机器人支持在一次回复中发送多条消息（在回复文本中插入 `[MSG]` 分隔符），模拟真人分批发送长消息的习惯。

## 表情反应

机器人可以对消息添加 Emoji 表情反应（reaction），进一步增强互动感。

## 命令列表

| 命令 | 说明 |
|------|------|
| `/start` | 开始使用机器人 |
| `/fortune` | 查看今日运势 |
| `/guesshistory` | 开始看图猜时代游戏 |
| `/muteme` | 让机器人忽略我的消息 |
| `/unmuteme` | 让机器人恢复回复我 |
| `/quiet` | 安静模式（不主动插话）|
| `/unquiet` | 恢复主动插话 |
| `/topic` | 进入话题追踪模式 |
| `/notopic` | 退出话题追踪模式 |
| `/about` | 关于机器人 |
| `/admin` | 管理员面板（管理员专用）|
| `/gadmin` | 群管理面板（群管理员专用）|
| `/settings` | Business 设置（私聊）|
